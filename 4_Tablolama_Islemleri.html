Sayfa üzerinde bir takım verileri tablo üzerinde göstermek iiçn size sunulan element "table" elementidir. "tr", "td", "th" şeklinde üçe ayrılır.

TD(table data) => Kolon bilgisini, TH(table header) => Başlık bilgisini, TR(table row) => Satır bilgisini temsil eder.

Bazı Öznitelikleri;

  • Width => Tablonun genişliğini verir
  • Border => Tablonun iç ve dış çerçeveleri için boyut verir.
  • Cellspacing => Hücrelerin birbirine olan uzaklıklarını verir.
  • Cellpadding => Hücre içerisindeki değerlerin dış çerçeveye olan uzaklıklarını verir
  • Colspan => Hücrelerin merge edilmesini sağlayan özniteliktir. Aşağıdaki tabloda vermiş olduğumuz 2 değeri o hücrenin 2 sütunlukmuş gibi davranmasını söyler.
  • Rowspan => Satırların merge edilmesini sağlar.
